    # 04-绝望少女拯救游戏

    # 作者：红月（刺猬猫已下架）

    # 状态：完结（约300章）

    # 类型：类西幻轻小说

    # 亮点：

    拯救绝望少女

    # 男主：

    江良：亚修·艾克、齐良、小白等等，开hg这方面比较主动，前往副本是身穿(后文有解释)。

    # 女主角：

    萝琳(林洛洛)：魔女。男主第一个世界拯救对象，其因魔女身份而遭人们唾弃，后投靠男主被男主的温暖打动并驱除了心中的黑暗。结尾男主挖心助其成为魔女之王，彻底白给。

    夏娅(夏青月)：精灵二公主、是其母亲的分身但有自己意识，男主第二个世界拯救对象。因为没有能力而无比自卑，实际上是其“母亲”用来作为召唤冥神的容器，男主发现后提着刀砍炸了冥界。男主付出代价消亡后将剑交付给她，成为冥府剑神。

    齐月霄(夜流月)：帝国皇女，男主第三个世界拯救对象。男主“青梅竹马”，实力强悍。但实际上是要被皇族献祭的祭品，在她得知后心灰意冷只有男主和齐雨瑶愿意陪伴在她身边。后爱上男主。最后男主帮助她成为了神龙女帝，并采用假死手段进一步增进感情。

    伊莉丝白(爱西丝|哈卡迪)：天使长，创天使。男主第四个世界拯救对象。占有欲极强的男主母亲，拥有两个分身。因为女儿的背叛而变得神经病。后在男主帮助下发誓只宠爱男主这一个孩子，发誓完后男主引导世间冤魂消灭自己而拯救了她。

    雪月花：冥月女神，小月亮。男主第四个世界(一万年前)拯救对象。在一万年前纯洁的她被男主一本恋爱小说染上了男主的颜色。一直等着男主等了一万年。

    苏雅娜：太阳女神，男主女儿。被男主撩过。在男主要求下保护一直雪月花。同样等了男主一万年。

    维利亚：艾克家族女仆长，算是拯救对象之一。幼时被父亲虐待后被男主救下。成为了男主家族中的女仆长，一直喜欢男主。

    其他女主：耶荷(荷月儿)；银月精灵女王和长公主；齐雨瑶(齐月霄姐妹)；艾克家族全体魔女、女仆。

    # 郁闷：

    1-有些女主曾遭虐待，没有详细描写，正常的殴打

    2-前期推了耶荷和女仆长，其他的都是结局推

    # 读后感（2024/2/16）：

    前两个世界好看，后面就有些审美疲劳了，总体上算是粮草。
