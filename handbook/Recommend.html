    # 01-恋爱要在模拟后

    # 作者：尺间萤火/归来细雨中（起点）

    # 状态：完结（368章+6番外）

    # 类型：类东京轻小说、模拟器

    # 亮点：感情细腻、文笔优秀、拯救

    # 男主：

    南悠希：有恋爱模拟器，前三个模拟中忠贞不二，后面开hg

    # 女主角（12女主+6女儿+全双C）：

    1-浅野奈绪：宅女，“富士山”社恐笨蛋大姐姐，家里蹲，温柔弱受===女儿：六和美羽

    2-伊吹夕子：“人偶般的娇小与可爱”，“夕子妈妈无微不至的照料”，只对男主有感情===女儿：真由理

    3-中岛玲奈：“妃殿下”，“病美人”===女儿：茉夏

    4-朝雾美月：金毛辣妹，把自己塑造为无情感者

    5-观月镜：武力代表，社畜

    6-南小鸠：调皮活泼，梦想变得南琉璃那样文静

    7-南琉璃：文静乖巧，梦想变得南小鸠那样活泼

    8-南舞夏：中二，催眠===女儿：真冬

    9-千寿一美：“女配”，“工具人===女儿：十花

    10-一之濑诗织：“首相”，“智慧担当===女儿：秋乃

    11-南心爱：妹妹，受气包，活泼，想哥哥一直留在自己身边

    12-朝雾茉优：养女，为父亲的感情奔波

    # 郁闷：

    1-有胃疼情节，不过我喜欢

    2-没有开车情节，尺度小

    # 读后感（2023/12）：

    这本书的前面三个模拟：奈绪、夕子、玲奈开局即巅峰；在我看过的所有小说中文笔感情也是最好的；中期女儿篇开始就正常水平了，稍有缺陷。总体而言，光前面三个模拟就推荐一看。一场意犹未尽的旅途。

    # 02-凡人之长生仙道

    # 作者：三驴大夫（起点）

    # 状态：完结（733章）

    # 类型：凡人修仙传同人

    # 亮点：爽，女主多

    # 读后感（2023/8/16）：

    女主见链接：http://tieba.baidu.com/p/8376661792

    我详看了642章，共732章，跳看了结局。这本书感情占比较少，所以没那么多郁闷，感情不怎么出彩。我个人比较喜欢看修仙小说，看这本书感觉还不错，有那种废寝忘食的感觉了，很爽。

    这本书我分为三个部分【血色禁地副本以前】【血色禁地副本到广寒宫副本】【广寒宫副本到结局】

    1. 【血色禁地副本以前】很劝退，男主人设的问题，“不主动 不承诺 不负责”，幸好后面男主性格转变了。 2. 【血色禁地副本到广寒宫副本】最爽的部分，情节还是很不错的，手不释卷。 3. 【广寒宫副本到结局】
    • 这部分里面【杀天奎狼王左右的情节】用了很多配角的言语来衬托主角牛p的情节，有反派yy女主的情节
    • 这部分里面【一人得道鸡犬升天】写的太尴尬了
    • 我看到642，还有一百章结局，通过我看结尾，还有1个女主没出现，会漏至少一个女配

    总结：第二部分是这本书的精华所在，建议看到【广寒宫副本】就不要再看了。642章主角又开始搞势力了,主角炼制一个天岛，让自己势力的人住到上面，我本来猜想只有女主们和主角住。我很反感搞势力这个事，没什么必要。其实从【广寒宫副本】以后就看得出作者急了，后面写的太赶了，观感下降。跳看结局之后，只能说这个结局马马虎虎。差不多就是这样。

    女主们双C，周媛是rq 紫灵她妈，我本来以为她不是女主，看结尾才发现她最后也收了。这本书感情描写比较少，但是贯穿始终，最后的结局也是感情类型的结局。女主们其实没什么郁闷。客观上我觉得这本书还是蛮不错的

    来源：http://tieba.baidu.com/p/8554895670

    # 04-这些大人物其实都是我

    # 作者：万里万花（刺猬猫）

    # 状态：完结（507章+5番外）

    # 类型：类西幻轻小说

    # 简介：

    我穿越了，穿越到没有魔法，没有斗气，甚至连电都没有的落后世界，还是标准的孤儿开局。 所幸的是，穿越给了我一个还算安慰的福利，那就是我能够在梦里体验各种奇幻生活。 在梦里，我曾是远古众神之一，带头掀起对原初的崇高叛乱。 在梦里，我曾是古老帝国最具权威的大公，一手开创这个伟大的帝国。 在梦里，我曾是神话时代最强大的英雄，所有邪恶都在为我胆寒。 在梦里，我曾是精灵的领袖，守护双圣树的最古王者。 在梦里，我曾是少女的神袛，但我却给她开了一个莫大的玩笑。 每个时代都能找到我的踪迹和传说，但却无人知晓这些鼎鼎大名者，皆是一人。 然后你现在突然告诉我，那其实不是梦，是真的？ 而且我招惹的那么多女神、圣女、公主、女皇等等还特么都活着？

    # 亮点：

    作品中设定：神与王的关系高于一切。神与王可以是伴侣、父子、兄弟等等，但比这些关系还要更进一步，拥有紧紧纠缠的因果，超越生死的感情。对于神来说，王是祂感情的全部，神不爱世人、不爱信徒、不爱与王的孩子，只爱王，王是神的锚点，失去王的神要么崩坏要么趋于无情。以众神对王的痴迷，反而王比神更重要（因为你赞美神，神不会高兴，但你赞美王，神会高兴，故而以男主为王的几个神都会送几个“天使”给男主）。这既是创世主(原初)的赐福，也是诅咒。

    # 男主：

    莫佐：最初的长子-崇高、克伦威尔大公、太阳王、画影之主、蔷薇王、人王阿拉贡、风暴王，都是他的马甲。一开始是个学生，打算过普通日子，卷入超凡事件后发现梦里的马甲是真实存在的历史，于是走上还情债的道路。

    # 女主角：

    【原初】：创世主，一体三魂，众神之父，世界的创造者，真正意义上的唯一女主。

    【奥黛丽】：学院三大玫瑰之一，后成为男主的财务官，笨笨的。

    【凯瑟琳】：温莎大公的女儿，憧憬浪漫，甜美活泼，称呼男主为我的骑士，被男主所救后对男主产生好感。

    【茉莉】：温莎大公的女儿，凯瑟琳表姐，性格认真，序列五的凡人巅峰，原为凯瑟琳的骑士，后成为男主的守护骑士。

    【薇儿丹尼】：北境第一美人，序列五的凡人巅峰，男主为克伦威尔大公时期的守护骑士的后代，是男主的血契骑士。（比夫妻更为亲近的关系，生死与共）

    【伊莎拜尔】：洛兰帝国女皇、光荣之主，300年前由克伦威尔为其扫清一切阻碍，成为序列一，后男主助其击败敌人成神，希望男主成为她的王，被男主当初拒绝，回溯时间徐徐图之。男主一直是以陛下和臣子的关系看待她，他们的关系在大贵族会上才真正开始转变为神与王的关系。

    【维多利亚】：旧维多利亚王朝画影之主-安格尔的女王，以及新维多利亚王朝的神，以梦魇一途成神，将安格尔变成她的王。独守画影之主漫长岁月，困于梦中不愿苏醒。

    【拉】：精灵王朝的暗影太阳，贤妻良母型，精灵王朝的神是【双圣树】，王是太阳王，而拉是受到双圣树眷顾的精灵。代替太阳王主持精灵的漫长岁月。

    【伊露维塔】：胜利女神，王是阿拉贡，在与莱雅女神斗争中，由于失去她的王而处于下风，找回阿拉贡之后便战胜了失去王的莱雅。

    【阿尔纹】：胜利女神的大天使长，序列一，胜利女神送给阿拉贡的女人。

    【贝尔娜黛】：正义女神，第一章被提尔神背刺的王，接替提尔神而成为了神，由于失去了提尔，导致正义愈发无情，提尔的回归让她逐渐找回感情。

    【海薇丝】：海洋女神，苦等属于她的风暴王漫长岁月，自暴自弃，对于信徒们的要求完全回应不加干涉。

    【奥菲莉儿】：海洋女神的猩红天使，同样由于风暴王的离去而心智接近崩溃。海洋女神送给风暴王的天使。

    【卡西亚】：接替奥菲莉儿成为天使长。海洋女神送给风暴王的天使。

    【拉妮】：人偶之神，原初送给崇高的人偶，当着众多女神的面，掳走她们的王-男主，男主也是她的蔷薇王。

    【艾希卡亚】月神，原初的长子之一，真正的神灵，喜欢自己的哥哥崇高同时也把他视作自己的王。

    # 郁闷：

    1-作者修罗场的描述单一，男主一直流冷汗

    2-车速为零、男主草食、一个未推

    3-伊莎拜尔女皇会让臣子亲吻她的手背，戴手套；男主是直接亲的

    4-胜利女神篇章，男主的义妹背叛了他的哥哥和男主，男主在杀她这件事上面显得过于仁慈(个人看法)

    5-大贵族会上，【双生树】的描写有点毁人设，转变太大，占了拉、艾希卡亚的便宜

    # 读后感（2024/2/1）：

    一场盛大的歌剧，虽然有一些小瑕疵，但是总体上还是给我带来了一场不错的阅读旅途。
